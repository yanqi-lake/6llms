# 解题思路

## 第一步：理解问题，从最简单的情形入手

### 最简单实例：K=1
- 只有一天股价，范围 [1, N]
- 答案显然是 N

### K=2 的情况
- 第一天股价 a1 ∈ [1, N]
- 第二天股价 a2，满足 a2 > a1 且 a2 - a1 ≤ M
- 枚举可得：答案 = M*N - M*(M+1)/2（当 M < N-1 时）

## 第二步：联想类似问题，形式化建模

将 K 天的股价记为 a1, a2, ..., aK，满足：
- 1 ≤ a1 < a2 < ... < aK ≤ N
- 差值 di = a(i+1) - ai ∈ [1, M]，i = 1 到 K-1

令 S = d1 + d2 + ... + d(K-1)，表示总增量。

对于给定的差值序列 (d1, d2, ..., d(K-1))：
- 第一天股价 a1 可以取 1 到 N-S，共 N-S 种
- 所以答案 = sum_{所有差值序列} (N - S)
