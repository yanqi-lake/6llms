题目标题: Second Best

题目描述:
You are given an integer sequence A=(A_1,...,A_N) of length N. Here, A_1, A_2, ..., A_N are all distinct.\nWhich element in A is the second largest?\n\nSample Input 1:\n4\n8 2 5 1\nSample Output 1:\n3\nThe second largest element in A is A_3, so print 3.\n\nSample Input 2:\n8\n1 2 3 4 5 10 9 11\nSample Output 2:\n6

输入格式:
The input is given from Standard Input in the following format:\nN\nA_1 A_2 ... A_N

输出格式:
Print the integer X such that the X-th element in A is the second largest.

样例输入:
4\n8 2 5 1

样例输出:
3

约束条件:
2 ≤ N ≤ 100\n1 ≤ A_i ≤ 10^9\nA_1, A_2, ..., A_N are all distinct.\nAll input values are integers.
