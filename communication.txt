## 题目描述

小 T 最近在学着买股票，他得到内部消息：F 公司的股票将会疯涨。股票每天的价格已知是正整数，并且由于客观上的原因，最多只能为 $N$。在疯涨的 $K$ 天中小 T 观察到：除第一天外每天的股价都比前一天高，且高出的价格（即当天的股价与前一天的股价之差）不会超过 $M$，$M$ 为正整数。并且这些参数满足 $M(K-1)<N$。小 T 忘记了这 $K$ 天每天的具体股价了，他现在想知道这 $K$ 天的股价有多少种可能。

## 输出格式

仅包含一个数，表示这 $K$ 天的股价的可能种数对于 $P$ 的模值。

## 输入输出样例 #1

### 输入 #1

```
7 3 2 997

```

### 输出 #1

```
16

```

# 解题思路

## 问题理解

给定参数 N、K、M、P，求满足以下条件的 K 天股价序列数量：
- 股价是正整数，不超过 N
- 从第二天起，每天股价比前一天高
- 每天的增长量（差值）在 [1, M] 范围内
- 满足 M(K-1) < N（这个条件确保所有差值组合都合法）

## 第一步：数学建模

设 K 天的股价为 $a_1, a_2, ..., a_K$，令 $d_i = a_{i+1} - a_i$（$i = 1$ 到 $K-1$），则：
- 每个 $d_i \in [1, M]$
- 差值序列有 $M^{K-1}$ 种可能
- 对于给定的差值序列，设 $S = \sum_{i=1}^{K-1} d_i$，则 $a_1$ 可以取 $1$ 到 $N-S$，共 $N-S$ 种取值

因此，总方案数为：
$$\sum_{d_1=1}^{M} ... \sum_{d_{K-1}=1}^{M} (N - \sum_{i=1}^{K-1} d_i)$$

## 第二步：公式化简

由于 $M(K-1) < N$，所有差值组合都满足条件（因为 $S \leq M(K-1) < N$）。

设 $f(S)$ 为差值和为 $S$ 的组合数，则：
$$\text{答案} = \sum_{S} f(S) \cdot (N - S) = N \cdot \sum_{S} f(S) - \sum_{S} S \cdot f(S)$$

其中 $\sum_{S} f(S) = M^{K-1}$（所有差值序列的总数）。

对于 $\sum_{S} S \cdot f(S)$，利用期望的线性性质：
- 每个 $d_i$ 的期望值为 $\frac{1+M}{2}$
- $S$ 的期望值为 $(K-1) \cdot \frac{1+M}{2}$
- 因此 $\sum_{S} S \cdot f(S) = M^{K-1} \cdot (K-1) \cdot \frac{1+M}{2}$

最终公式：
$$\text{答案} = M^{K-1} \cdot N - M^{K-1} \cdot (K-1) \cdot \frac{M+1}{2}$$

## 第三步：算法设计

1. **快速幂计算**：使用快速幂算法计算 $M^{K-1} \pmod P$
2. **乘法与除法**：
   - 计算 $A = M^{K-1} \pmod P$
   - 计算 $B = (K-1) \times (M+1) \pmod P$
   - 计算答案 $= (N \pmod P) \times A - A \times B \times \text{inv}(2) \pmod P$
3. **求逆元**：使用扩展欧几里得算法求 2 在模 P 下的逆元（注意 P 可能不是质数）

## 第四步：复杂度分析

- 时间复杂度：$O(\log K)$（快速幂）
- 空间复杂度：$O(1)$

## 验证示例

输入：N=7, K=3, M=2, P=997
- $M^{K-1} = 2^2 = 4$
- $(K-1) \times (M+1) / 2 = 2 \times 3 / 2 = 3$
- 答案 = $4 \times 7 - 4 \times 3 = 28 - 12 = 16$
- 16 mod 997 = 16 ✓

## 关键点总结

1. 利用期望的线性性质简化计算，避免枚举所有差值组合
2. 条件 $M(K-1) < N$ 确保所有差值组合都合法，简化了求和范围
3. 使用扩展欧几里得算法处理模 P 下的除法（不需要 P 是质数）

typedef long long ll;

ll mod_pow(ll base, ll exp, ll mod) {
    ll result = 1;
    base %= mod;
    while (exp > 0) {
        if (exp & 1) result = (result * base) % mod;
        base = (base * base) % mod;
        exp >>= 1;
    }
    return result;
}

ll mod_inv(ll a, ll mod) {
    ll m0 = mod, t, q;
    ll x0 = 0, x1 = 1;
    if (mod == 1) return 0;
    while (a > 1) {
        q = a / mod;
        t = mod;
        mod = a % mod;
        a = t;
        t = x0;
        x0 = x1 - q * x0;
        x1 = t;
    }
    if (x1 < 0) x1 += m0;
    return x1;
}

int main() {

    cin >> N >> K >> M >> P;

    ll inv2 = mod_inv(2, P);
    ll term1 = (N % P * A) % P;
    ll term2 = (A * B % P * inv2) % P;
    ll result = (term1 - term2 + P) % P;

}

## 问题理解

给定参数 N, K, M, P，求满足以下条件的 K 天股价序列数量：
- 每天股价为正整数，不超过 N
- 从第 2 天起，每天比前一天高，且增量在 [1, M] 范围内

## 第一步：理解最简单实例

**K = 1 的情况**：只有 1 天，股价可以是 1 到 N 的任意值，答案为 N。

**K = 2 的情况**：
- 第 1 天价格 a，第 2 天价格 b，满足 a < b ≤ a+M ≤ N
- 枚举可得：答案 = Σ_{S=1}^{M} (N-S)，其中 S = b-a
- 计算得：答案 = M×N - M(M+1)/2

## 第二步：联想与公式推导

这个问题本质是**受限整数分拆**：将 K-1 个正整数（每个在 1 到 M 之间）进行组合。

设增量序列为 d_2, d_3, ..., d_K，每个在 [1, M] 之间，总增量 S = Σd_i。

- 第一天股价 a_1 可以取 1 到 N-S，共 N-S 种
- 增量序列有 M^{K-1} 种

利用生成函数或求导计算：
- Σ g(S) = M^{K-1}（总序列数）
- Σ S×g(S) = (K-1) × M^{K-1} × (M+1)/2（增量之和）

因此：
$$\text{答案} = N \times M^{K-1} - \frac{(K-1) \times M^{K-1} \times (M+1)}{2}$$

化简为：
$$\text{答案} = \frac{M^{K-1} \times (2N - (K-1)(M+1))}{2}$$

## 第三步：取模处理

由于 P 可能不是质数，不能直接使用逆元。采用以下方法处理除以 2：

利用数学恒等式：对于整数 a, b，若 a×b 是偶数，则：
$$\frac{a \times b}{2} \pmod P = \frac{(a \times b) \bmod (2P)}{2} \pmod P$$

因此：
1. 计算 A = M^{K-1} mod (2P)
2. 计算 B = (2N - (K-1)(M+1)) mod (2P)
3. 计算 X = (A × B) mod (2P)
4. 答案 = (X / 2) mod P

- **算法**：快速幂 + 取模运算
- **时间复杂度**：O(log K)，因为需要计算 M^{K-1}
- **空间复杂度**：O(1)

# 解题思路

## 问题理解

给定参数 N, K, M, P，其中：
- 股票价格是正整数，不超过 N
- 观察 K 天，满足：每天比前一天高，且增量在 [1, M] 之间
- 条件 M(K-1) < N 保证解的存在性

求满足条件的 K 天股价序列数目，答案对 P 取模。

## 第一步：数学建模

设第 i 天股价为 $a_i$，第 i 天与第 i-1 天的差为 $d_i = a_i - a_{i-1}$（i ≥ 2）。

则：
- $d_i \in [1, M]$
- 总增量 $S = \sum_{i=2}^{K} d_i = a_K - a_1$
- 对于固定的增量序列，$a_1$ 可以取 $1$ 到 $N-S$，共 $N-S$ 种

因此总数 = $\sum_{d_i \in [1,M]} (N - \sum d_i)$

## 第二步：公式简化

令 $T = \sum_{d_i \in [1,M]} \sum_i d_i$

总数 = $N \cdot M^{K-1} - T$

由于对称性，$T = (K-1) \cdot M^{K-2} \cdot \sum_{d=1}^{M} d = (K-1) \cdot M^{K-2} \cdot \frac{M(M+1)}{2}$

代入得：
$$总数 = M^{K-1} \cdot N - (K-1) \cdot M^{K-1} \cdot \frac{M+1}{2}$$

$$= M^{K-1} \cdot \frac{2N - (K-1)(M+1)}{2}$$

## 第三步：处理模运算

由于 P 不一定是质数，无法直接求逆元。采用以下技巧：

1. 在 $2P$ 模下进行计算（因为最后要除以 2）
2. 设 $X = 2N - (K-1)(M+1)$（可能在负数范围，需调整）
3. 计算 $power = M^{K-1} \pmod{2P}$
4. 计算 $result = (power \times X) \pmod{2P}$
5. 由于答案整数，result 必为偶数，令 $result = result / 2$
6. 最终答案 $= result \pmod P$

## 第四步：算法实现

- 使用快速幂计算 $M^{K-1} \pmod{2P}$，时间复杂度 $O(\log K)$
- 所有乘法使用 128 位整数（__int128）避免溢出
- 边界处理：负数取模时加 $2P$ 转为正数

    ll X = (2 * (N % mod) % mod - mul_mod(K - 1, M + 1, mod) + mod) % mod;
    ll result = mul_mod(power, X, mod);
    result = (result % mod + mod) % mod / 2;
    cout << result % P << endl;

    return 0;
}
