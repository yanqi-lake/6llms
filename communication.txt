# P3228 [HNOI2013] 数列

## 题目描述

小 T 最近在学着买股票，他得到内部消息：F 公司的股票将会疯涨。股票每天的价格已知是正整数，并且由于客观上的原因，最多只能为 $N$。在疯涨的 $K$ 天中小 T 观察到：除第一天外每天的股价都比前一天高，且高出的价格（即当天的股价与前一天的股价之差）不会超过 $M$，$M$ 为正整数。并且这些参数满足 $M(K-1)<N$。小 T 忘记了这 $K$ 天每天的具体股价了，他现在想知道这 $K$ 天的股价有多少种可能。

## 输入格式

只有一行用空格隔开的四个数：$N$、$K$、$M$、$P$。对 $P$ 的说明参见后面“输出格式”中对 $P$ 的解释。输入保证 $20\%$ 的数据 $M,N,K,P \le 20000$，保证 $100\%$ 的数据 $M,K,P \le 10^9$，$N \le 10^{18}$。

## 输出格式

仅包含一个数，表示这 $K$ 天的股价的可能种数对于 $P$ 的模值。

## 输入输出样例 #1

### 输入 #1

```
7 3 2 997

```

### 输出 #1

```
16

```

## 说明/提示

**样例解释**  

输出样例的 $16$ 表示输入样例的股价有 $16$ 种可能：

$\{1, 2, 3\}$，$\{1, 2, 4\}$，$\{1, 3, 4\}$，$\{1, 3, 5\}$，$\{2, 3, 4\}$，$\{2, 3, 5\}$，$\{2, 4, 5\}$，$\{2, 4, 6\}$，$\{3, 4, 5\}$，$\{3, 4, 6\}$，$\{3, 5, 6\}$，$\{3, 5, 7\}$，$\{4, 5, 6\}$，$\{4, 5, 7\}$，$\{4, 6, 7\}$，$\{5, 6, 7\}$。

# 解题思路

## 问题分析

这道题目要求计算满足以下条件的K天股价序列数量：
1. 股价是正整数，最大为N
2. 每天股价严格递增
3. 相邻两天股价差不超过M
4. 满足 M(K-1) < N

## 第一步：理解最简单情形

**K = 1 的情况：**
- 只有一天，股价可以是 1 到 N 中的任意值
- 答案 = N

**K = 2 的情况：**
- 第一天股价为 a₁，第二天为 a₂
- 约束：a₁ < a₂ ≤ N，且 a₂ - a₁ ≤ M
- 对于每个差值 d = a₂ - a₁（1 ≤ d ≤ M），a₁ 有 N-d 种取值
- 答案 = Σ_{d=1}^{M} (N-d) = M·N - M(M+1)/2

**K = 3 的情况（验证）：**
- 设第二天相对第一天增加 d₂，第三天相对第二天增加 d₃
- 每个增量取值 1 到 M，共 M² 种组合
- 对于每组增量，总增量为 S = d₂ + d₃
- 起始股价有 N-S 种可能
- 答案 = Σ_{d₂=1}^{M} Σ_{d₃=1}^{M} (N - d₂ - d₃)

## 第二步：推广到一般情况

**引入增量表示：**
- 设第 i 天的增量为 dᵢ = aᵢ - aᵢ₋₁（i ≥ 2）
- 每个 dᵢ 满足 1 ≤ dᵢ ≤ M
- 总增量 S = Σ_{i=2}^{K} dᵢ

**关键约束：**
- 由于 M(K-1) < N，而 dᵢ ≤ M
- 所以 S ≤ M(K-1) < N
- 这意味着对于所有增量组合，N - S 始终为正

**计数公式：**
- 增量序列共 M^{K-1} 种
- 每种序列贡献 N - S
- 答案 = Σ (N - S) = M^{K-1} · N - Σ S

**计算 ΣS：**
- 每个 dᵢ 在所有序列中出现 M^{K-2} 次
- ΣS = M^{K-2} · (K-1) · Σ_{j=1}^{M} j
- Σ_{j=1}^{M} j = M(M+1)/2

**最终公式：**
- K = 1: 答案 = N
- K ≥ 2: 答案 = M^{K-2} · [M·N - (K-1)·M(M+1)/2]

## 第三步：验证

以样例 N=7, K=3, M=2 为例：
- M^{K-2} = 2¹ = 2
- M·N = 14
- (K-1)·M(M+1)/2 = 2 × 2 × 3 / 2 = 6
- 答案 = 2 × (14 - 6) = 2 × 8 = 16 ✓

## 第四步：算法实现

**计算步骤：**
1. 处理 K = 1 的特殊情况
2. 计算 M^{K-2} mod P（使用快速幂）
3. 计算 M·N mod P
4. 计算 (K-1)·M·(M+1)/2 mod P（注意使用128位整数防止溢出）
5. 计算括号内的差值 mod P
6. 乘积取模得到最终答案

**注意事项：**
- P 可能不是质数，不能使用逆元，直接使用除法（因为除的是2，结果是整数）
- N ≤ 10^18，M, K, P ≤ 10^9，乘法可能溢出64位，需要使用128位整数或快速乘模算法

typedef long long ll;

ll mod_pow(ll base, ll exp, ll mod) {
    ll result = 1;
    base %= mod;
    while (exp > 0) {
        if (exp & 1) result = (result * base) % mod;
        base = (base * base) % mod;
        exp >>= 1;
    }
    return result;
}

ll mod_mul(ll a, ll b, ll mod) {
    ll result = 0;
    a %= mod;
    while (b > 0) {
        if (b & 1) result = (result + a) % mod;
        a = (a + a) % mod;
        b >>= 1;
    }
    return result;
}

typedef long long ll;
typedef __int128 i128;

ll N, K, M, P;

ll mul_mod(ll a, ll b, ll mod) {
    return (ll)((i128)a * b % mod);
}

ll pow_mod(ll base, ll exp, ll mod) {
    ll result = 1;
    while (exp > 0) {
        if (exp & 1) result = mul_mod(result, base, mod);
        base = mul_mod(base, base, mod);
        exp >>= 1;
    }
    return result;
}

int main() {
    cin >> N >> K >> M >> P;
    N %= P;

    if (K == 1) {
        cout << N % P << endl;
        return 0;
    }

    ll part1 = mul_mod(M % P, N % P, P);
    ll sum_d = mul_mod(M % P, (M + 1) % P, P);
    sum_d = mul_mod(sum_d, ((K - 1) % P), P);
    sum_d = (sum_d / 2) % P;

    ll bracket = (part1 - sum_d + P) % P;
    ll power = pow_mod(M % P, K - 2, P);
    ll ans = mul_mod(power, bracket, P);

    cout << ans << endl;

}

int64 mod_pow(int64 a, int64 e, int64 mod) {
    int64 res = 1 % mod;
    a %= mod;
    while (e > 0) {
        if (e & 1) res = mod_mul(res, a, mod);
        a = mod_mul(a, a, mod);
        e >>= 1;
    }
    return res;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int64 N, K, M, P;
    if (!(cin >> N >> K >> M >> P)) return 0;
    
    if (P == 1) {
        cout << 0 << "\n";
        return 0;
    }
    
    if (K == 1) {
        cout << (N % P) << "\n";
        return 0;
    }
    
    int64 powM = mod_pow(M % P, K - 2, P);
    int64 term1 = mod_mul(M % P, N % P, P);
    int64 sum = M % P * ((M + 1) % P) % P;
    sum = sum * ((P + 1) / 2) % P;
    int64 term2 = mod_mul((K - 1) % P, sum, P);
    int64 diff = (term1 - term2) % P;
    if (diff < 0) diff += P;
    int64 result = mod_mul(diff, powM, P);
    
    cout << result << "\n";
    return 0;
}
